## Gmail Automation Features You Need to Know

### 1. Canned Responses (Templates)

Save and reuse email templates:

**Setup:**
1. Go to Settings → See all settings
2. Enable "Templates" in Advanced
3. Compose your template email
4. Click three dots → Templates → Save as template

**Best uses:**
- Common questions and answers
- Meeting confirmations
- Status updates
- Standard responses

### 2. Filters and Labels

Automatically organize incoming email:

**Setup:**
1. Click the filter icon in search
2. Set criteria (sender, subject, keywords)
3. Click "Create filter"
4. Choose actions (label, archive, forward, etc.)

**Best uses:**
- Sort newsletters
- Prioritize VIP senders
- Archive notifications
- Flag urgent items

### 3. Vacation Responder

Auto-reply when unavailable:

**Setup:**
1. Settings → See all settings
2. Scroll to "Vacation responder"
3. Set dates and message
4. Enable

**Pro tips:**
- Set clear expectations
- Include alternative contact
- Mention return date
- Keep it professional

### 4. Undo Send

Prevent email mistakes:

**Setup:**
1. Settings → See all settings
2. Find "Undo Send"
3. Set cancellation period (5-30 seconds)

**Saves you from:**
- Forgotten attachments
- Wrong recipients
- Typos in important emails
- Sending before ready

### 5. Smart Compose

AI-powered writing assistance:

**Features:**
- Predictive text suggestions
- Grammar corrections
- Tone suggestions
- Quick phrases

**Enable:** Settings → General → Smart Compose

## Gmail Automation Best Practices

### 1. Keep It Personal

Automation shouldn't feel robotic:

- Use recipient names
- Reference specific details
- Maintain your voice
- Add personal touches when possible

### 2. Don't Over-Automate

Find the right balance:

**Good automation:**
- Routine responses
- Meeting confirmations
- Status updates
- Follow-up reminders

**Keep manual:**
- Sensitive conversations
- Complex negotiations
- Relationship building
- Unique situations

### 3. Test Your Automations

Before going live:

- Send test emails to yourself
- Check all links and attachments
- Verify personalization works
- Review timing and triggers

### 4. Monitor and Optimize

Track what's working:

- Open rates on automated emails
- Response rates
- Unsubscribe/complaint rates
- Time saved

### 5. Respect Recipients

Be transparent and helpful:

- Don't hide that it's automated
- Provide easy opt-out
- Keep frequency reasonable
- Always provide value
